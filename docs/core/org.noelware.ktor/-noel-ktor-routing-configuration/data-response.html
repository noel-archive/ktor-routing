@ExperimentalApi
fun dataResponse(callee: DataResponseCallee<Result<Any>, Exception?>): NoelKtorRoutingConfiguration

Sets the DataResponseCallee to when kotlin.Result is supported when returned by any endpoint method. This can be only added if supportKotlinResult is true, otherwise you're fine by not using this. It is also recommended to use this method so the plugin knows how to handle the result, otherwise it'll throw away the result.

this configuration to chain methods
